## Scale Notes

- Pixel and rem values are documented together for implementation handoff.
- `aliasTo` preserves the source unit variable reference from Figma.
- The visible Storybook preview applies the pixel value as `border-radius` on the sample shape.
- `radius-full` intentionally maps to `9999px` for pill/circle shapes.
