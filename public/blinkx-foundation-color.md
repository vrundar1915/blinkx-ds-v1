# blinkX Design System V1 - Foundation Color

This markdown file documents the current Foundation / Color data used by the blinkX DS V1 Storybook color pages.

The values below are generated from the Storybook source files so the downloadable file matches the visible swatches:

- `src/foundation/color/primitiveColors.ts`
- `src/foundation/color/semanticColors.ts`

## Counts

| Collection | Light | Dark |
| --- | ---: | ---: |
| Primitive colors | 159 | 159 |
| Semantic colors | 238 | 238 |

## Naming

- Primitive color swatches use their short display label in Storybook, for example `brand-25` instead of `Colors/brand/brand-25`.
- The full Figma variable path is preserved in the markdown tables for traceability.
- Primitive colors are raw color values. They are not semantic tokens.
- Dark theme values are listed separately from light theme values.
