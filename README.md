# blinkX DS V1

Storybook documentation for the blinkX Design System V1.

## Run Locally

```bash
npm install
npm run storybook
```

Storybook runs at `http://127.0.0.1:6006`.

## Build

```bash
npm run build-storybook
```

The production Storybook build is configured for Vercel with `vercel.json`.
